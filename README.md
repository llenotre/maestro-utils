# About

maestro-utils provides utility commands for the Maestro operating system:
- User management:
	- su
	- passwd
	- useradd
	- userdel
	- usermod
	- groupadd
	- groupdel
	- groupmod
	- login
	- nologin
- Kernel utilities:
	- dmesg
	- mount
	- umount
	- insmod
	- rmmod
	- lsmod
- Filesystem utilities:
	- fdisk
	- sfdisk
	- mkfs
	- fsck
- System power:
	- halt
	- poweroff
	- shutdown
	- reboot
