# maestro-utils

maestro-utils provides utility commands for the Maestro operating system:
- User management:
	- su
	- passwd
	- useradd
	- userdel
	- usermod
	- groupadd
	- groupdel
	- groupmod
- Kernel utilities:
	- dmesg
	- mount
	- umount
	- insmod
	- rmmod
	- lsmod
- System power:
	- halt
	- poweroff
	- shutdown
	- reboot
